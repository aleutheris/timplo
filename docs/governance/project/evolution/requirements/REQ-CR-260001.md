# Requirement: REQ-CR-260001 Static hosting only

- Requirement ID: REQ-CR-260001
- Title: Static hosting only
- Class: CR
- Status: Active
- Owner: repository maintainer

## Statement

The application must remain a static web application and must not depend on a backend service.

## Rationale

Static-only is a hard product constraint that keeps hosting free and operations trivial; it bounds the architecture (no server, no stored user data off-device).

## Acceptance Criteria (behavioral)

- The app runs entirely in the browser.
- The app can be hosted on GitHub Pages.
- No server-side APIs are required for core functionality.
