# Requirement: REQ-FR-260005 Countdown visual state

- Requirement ID: REQ-FR-260005
- Title: Countdown visual state
- Class: FR
- Status: Active
- Owner: repository maintainer

## Statement

The active timer view must visibly distinguish between initial paused, running, and stopped paused states, and a finished timer must return to the same visual state as the initial paused state.

## Rationale

Clear visual states let users tell at a glance whether a timer is running, paused, or reset, without reading the number.

## Acceptance Criteria (behavioral)

- The initial paused state is visually distinct from the running state.
- The stopped paused state is visually distinct from the running state.
- A finished timer returns to the same visual state as the initial paused state.
- The countdown view remains readable while the timer is active.
