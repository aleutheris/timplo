# Requirement: REQ-FR-260004 Reset and finish behavior

- Requirement ID: REQ-FR-260004
- Title: Reset and finish behavior
- Class: FR
- Status: Active
- Owner: repository maintainer

## Statement

The application must let the user reset the selected timer to its configured duration, and when a timer reaches zero it must immediately return to its initial paused state so it can be started again with another tap.

## Rationale

Reset and auto-return-to-paused let a timer be reused immediately without re-configuration, which is the common repeat-timer workflow.

## Acceptance Criteria (behavioral)

- Reset returns the timer to its original configured time.
- Reset stops the timer if it is running.
- Reset is available below the selected timer in the selected timer view.
- When the countdown reaches zero, the timer returns to its initial paused state.
- Tapping a timer after it has finished starts the countdown again from its configured duration.
