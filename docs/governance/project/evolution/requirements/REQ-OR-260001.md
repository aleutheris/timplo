# Requirement: REQ-OR-260001 Static GitHub Pages deployment

- Requirement ID: REQ-OR-260001
- Title: Static GitHub Pages deployment
- Class: OR
- Status: Active
- Owner: repository maintainer

## Statement

The application must be deployable as a static site on GitHub Pages with no backend services.

## Rationale

Static GitHub Pages hosting keeps the app free to run, backend-less, and simple to deploy from the main branch.

## Acceptance Criteria (behavioral)

- The project builds into static files.
- The build output uses a relative base path suitable for GitHub Pages.
- A GitHub Actions workflow can publish the built site.
- The app runs in a browser without server-side rendering.
