# Requirement: REQ-FR-260001 Multi-timer countdown behavior

- Requirement ID: REQ-FR-260001
- Title: Multi-timer countdown behavior
- Class: FR
- Status: Active
- Owner: repository maintainer

## Statement

The application must let the user create multiple named timers, edit each timer's duration in minutes and seconds, select any timer into an enlarged active view, start or stop the selected timer by clicking or tapping it, and reset a timer to its configured duration.

## Rationale

Defines the core multi-timer behavior the product exists to provide: several countdowns the user can name, configure, run, and reset directly.

## Acceptance Criteria (behavioral)

- The user can add more than one timer.
- Each timer has a name and editable minute and second fields.
- Selecting a timer moves it into the active countdown view without starting it.
- Clicking or tapping the selected timer starts it when it is idle and stops or resumes it when appropriate.
- Reset returns the timer to its configured duration and stops it.
- The countdown shows minutes and seconds only.
