# Requirement: REQ-FR-260002 Multiple named timers

- Requirement ID: REQ-FR-260002
- Title: Multiple named timers
- Class: FR
- Status: Active
- Owner: repository maintainer

## Statement

The application must allow the user to create and manage multiple timers in a timer library view, and each timer must have a user-editable name and duration.

## Rationale

Users keep a set of reusable, independently named timers; the library view is where they add, name, and edit them.

## Acceptance Criteria (behavioral)

- The user can create more than one timer.
- Each timer has a distinct name.
- The user can edit each timer independently.
- Timer durations are configurable per timer.
- Delete is available in the timer library view only.
- Editing is available in the timer library view only.
